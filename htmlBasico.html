Introduccion HTML

Que es HTML?

  1. HTML o Hyper Text Markup Language es un lenguaje de marcas para crear Paginas Web
  2. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar el contenido

Breve Historia

  1. Creado por Tim Berners-Lee el 1991.
  2. HTML 2.0(1995): Primer estandard oficial.
  3. HTML5(2014): Última versión con nuevas funcionalidades como soporte para video, audio y graficos.







Estructura Básica

Estructura General

  1. La estructura de un documento HTML es jerárquica de etiquetas que definen las partes de una pagina.
  2. Cada página HTML sigue una estructura similar, comenzando por el <!DOCTYPE>, seguido de las etiquetas html,
    <head> y <body>.














Etiquetas Básicas

<title>

Define el título que aparece en las pestañas del navegador o en los resultados de búsqueda.

<h1> a <h6>

Utilizado para definir títulos y subtitulos en la página web (h1>h6).

<p>

Se utiliza para definir bloques de texto o párrafos.

<br>

Fuerza un salto de línea.

& nbsp;

Inserta un espacio en blanco que no se corta entre líneas.


















Elementos HTML y Anidamiento

Diferencia entre etiquetas y elementos

Etiqueta (tag)
  • Parte que define el principio y fin de un componente en HTML
  • Se delimita con los símbolos < >.
    Ejemplos:<p>,</p>,<h1>,</h1>.

Elemento
  • Un Elemento incluye la etiqueta de apertura y la de cierre.
  • Los elementos pueden ser vacios o contener otros elementos dentro.

Elementos Anidados

  • Anidamiento se refiere a colocar un elemento HTML dentro de otro elemento.
  • Es importante seguir el orden de etiquetas, la primera que se abre es la última que se cierra.





















Atributos HTML

Los atributos en HTML son elementos que se añaden a las etiquetas para modificar su comportamiento
o apariencia. Estos atributos se escriben en la etiqueta de apertura y se escriben tal que: nombre="valor".





Texto Formatado

<b> y <strong>: Negrita

  • <b> aplica negrita sin indicar importancia semantica

  • <strong> aplica negrita indicando importancia semantica



<i> y <em>: Cursiva

  • <i> aplica cursiva sin indicar enfasis

  • <em> aplica cursiva indicando enfasis



<u>: Subrayar

  • <u> subraya el texto



<strike> y <s>: Texto rayado

  • <strike/s> aplica un rayado al texto mostrando que esta eliminado o que es irrelevante



<sup>: Superíndice

  • <sup> aplica un superíndice que eleva el texto



<sub>: Subíndice

  • <sup> aplica un subíndice que baja el texto







Comentarios en HTML

  • Los comentarios en HTML son líneas de texto añadidal al código para explicar o anotar ciertas cosas pero son invisibles
    para el usuario final de la página.
  • Se utiliza para que los desarrolladores puedan documentar el código y dejar indicaciones a futuros programadores.
  • Estos no afectan ni al rendimiento ni a la visualización de la página.




Insercion de imagenes

  • La etiqueta <img> se utiliza para insertar imagenes en una página web
  • No tiene etiqueta de cierre ya que es un elemento auto-cerrado
  • Es importante proporcionar informacion sobre la imagen para que se muestre correctamente y mejorar la accesibilidad

Atributos de imagen

src(source)
  • Define la fuente o el camino de la imagen
  • Puede ser un camino relativo o una URL
alt(alternate text)
  • Proporciona un texto alternativo que se muestra si la imagen no carga
  • Es fundamental para la accesibilidad y el SEO.
width y height
  • Especifica la altura (height) o la anchura (width) de la imagen con píxeles o porcentages
  • Es importante para controlar la medida de las iagenes




Enlaces

  • la etiqueta <a> se utiliza para crear enlaces a otras páginas web, documentos, secciones en la misma página etc.
  • El elemento esencial para crear enlaces es la etiqueta href que define la URL donde llevará el enlace.
  • El texto dentro de el elemento es el que verán los usuarios.

Atributos de enlace

  1. href(Hipertext Reference):
    • Define la URL de destino.
    • Puede ser un enlace absoluto(url externa) o relativo(un camino en el mismo dominio).
  2. Target:
    • Define donde se abrirá la nueva pestaña
    • Hay dos tipos:
      • _blank: Se abrirá en una nueva pestaña
      • _self: Se abrirá en la misma pestaña




Listas

Liatas no ordenadas (<ul>): Liatas ordenadas (<ol>):
  1. Se utilizan para listas sin orden específico
  2. Los elementos se muestran con viñetas antes de cada item
  3. Cada elemento de la lista se identifica con la etiqueta <li>(list item)
  1. Se utilizan para listas donde importe el orden
  2. Los elementos se numeran automaticamente
  3. Cada elemento de la lista se identifica con la etiqueta <li>(list item)


Atributos de lista

  1. Tipo de atributo (type)

    Se utiliza para elegir el tipo de numeración

    • 1: Números (predeterminado)
    • A: Letras en mayúscula
    • a: Letras en minúscula
    • I: Números romanos en mayúscula
    • i: Numeros romanos en minúscula
  2. Tipo de atributo (type)

    Define el número o letra por el que empieza a numerar





Tablas

Etiqueta <table>:

  • La etiqueta <table> define la tabla

Etiqueta <tr>(table row):

  • Define una fila dentro de la tabla

Etiqueta <td>(table data):

  • Define una celda dentro de la tabla

Atributos de Tablas

  1. Border:
    • Añade un borde a la tabla y las celdas


  2. Cellpadding:
    • Define el espacio entre el contenido de la tabla y las celdas


  3. Cellspacing:
    • Controla el espacio entre las celdas


Etiquetas de Tablas

  • Etiqueta <thead>(table head):
    • Define el encabezamiento de la tabla, agrupandolas en filas de encabezamiento


  • Etiqueta <tbody>(table body):
    • Agrupa el cuerpo de la tabla. Se suele utilizar para separar los datos del encabezamiento


  • Etiqueta <th>(table header):
    • Se utiliza para crear una celda de encabezamiento en una tabla, el texto en este elemento aparece centrado y
      en negrita por defecto
